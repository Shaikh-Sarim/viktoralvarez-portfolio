4. Enjoy the smooth animations and modern design!

## Customization

### Colors
Edit the CSS variables at the top of `styles.css`:
```css
:root {
    --primary-color: #6c5ce7;
    --secondary-color: #00b894;
    --accent-color: #fdcb6e;
    /* ... more colors ... */
}
```

### Text Content
Update the text in `index.html` to match desired content.
